Como usar o endpoint:

Este endpoint mostra os jogos do dia baseado no torneio pesquisado. A atualização dos jogos e do placar é automática, porém pode levar algum tempo até que os dados sejam atualizados, não é algo instantâneo. A lista de torneio é voltada para os jogos onde os times brasileiros participam.

https://repl.c4ldas.com.br/api/valorant/schedule?channel=$(channel)&league=LEAGUE_NAME

Possíveis valores para LEAGUE_NAME:

League name Torneio
challengers_br Challengers BR
vct_lock_in VCT LOCK//IN
game_changers_series_brazil Game Changers BR
last_chance_qualifier_br_and_latam Last Chance Qualifier BR & Latam
vct_americas VCT Americas
vct_masters Masters
champions Champions
ascension_americas Ascension Americas
last_chance_qualifier_americas Last Chance Qualifier Americas
game_changers_championship Game Changers Championship
vct_emea VCT EMEA

Criando o comando manualmente para o Streamelements:

Para criar o comando no Streamelements, vá até o dashboard, crie um novo comando e coloque o seguinte na resposta (substitua LEAGUE_NAME pelo League name do quadro superior):

.me ${touser} ► ${customapi.https://repl.c4ldas.com.br/api/valorant/schedule?channel=$(channel)&league=LEAGUE_NAME}

Mais torneios podem ser adicionados, à medida que a Riot for criando eles. Caso tenha algum torneio que não esteja na lista, entre em contato comigo pelo Twitter, Twitch, Github, Instagram, qualquer lugar pra informar, o usuário é sempre @c4ldas
