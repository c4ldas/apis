Como usar o endpoint:

https://repl.c4ldas.com.br/api/valorant/rank?channel=$(channel)&player=USERNAME"&tag=TAGLINE&msg=$(queryescape "MENSAGEM")

https://repl.c4ldas.com.br/api/valorant/rank?channel=$(channel)&id="ID"&msg=$(queryescape "MENSAGEM")

Exemplo:

Se o seu nick for: LOUD Coreano#LLL
USERNAME = "LOUD Coreano"
TAGLINE = LLL
MENSAGEM (opcional) = Mensagem que será exibida no chat quando o endpoint for acessado. O conteúdo da mensagem deve ficar entre aspas " ".
Possíveis variáveis em MENSAGEM: (player), (pontos), (rank). Caso seja Imortal/Radiante, também pode usar (posicao) e (vitorias) para mostrar sua posição e quantidade de vitórias, respectivamente.
Caso não adicione nenhuma mensagem, a seguinte mensagem padrão será exibida (utilizando o usuário do exemplo):
LOUD Coreano tem 213 pontos e tá Imortal 2
ID = Para informações como usar o ID ao invés de player/tag, veja mais abaixo ou clique aqui: Usando ID

Configure a sua mensagem:

Utilize o quadro abaixo para preencher seu nick, tag e mensagem (opcional), depois clique em "Mostrar mensagem formatada".
Caso esteja satisfeito, clique em "Gerar código para chat"
Variáveis disponíveis: (player)  (rank)  (pontos)
Para Imortal/Radiante: (player)  (rank)  (pontos)  (posicao)  (vitorias)

Criando o comando manualmente para o Streamelements:

Para adicionar o comando, use o quadro superior para gerar o comando ou crie manualmente usando o exemplo abaixo:

Caso seu nick possua caracteres especiais, acentos ou coisas esquisitas (você tem problemas?), use o quadro superior ou prefira usar a opção de ID logo abaixo.

.me ${touser} ► ${customapi.https://repl.c4ldas.com.br/api/valorant/rank?channel=$(channel)&player="LOUD Coreano"&tag=LLL&msg=$(queryescape "(player) tem (pontos) pontos e está (rank), com (vitorias) vitórias e em (posicao)º na tabela")}

Usando ID ao invés de player/tag

Também é possível utilizar a opção id ao invés de player e tag. Com o ID, você não precisará ficar atualizando o comando cada vez que troca de nick (novamente, qual o seu problema?). O comando ficará assim:

.me ${touser} ► ${customapi.https://repl.c4ldas.com.br/api/valorant/rank?channel=$(channel)&id=10726a29-ce65-5471-a794-32733f309a16&msg=$(queryescape "(player) tem (pontos) pontos e está (rank), com (vitorias) vitórias e em (posicao)º na tabela")}

Para descobrir o seu id (também chamado de puuid), abra essa página e digite o seu nick e tag:

Encontre o seu ID (puuid)
